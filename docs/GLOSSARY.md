# Glossary

- **Capability** — an unforgeable token granting specific authority over a
  specific object. The only source of authority in Fjord.
- **CSpace / VSpace** — a task's capability space / virtual address space.
- **TCB** — Trusted Computing Base: code whose correctness security depends on.
- **DICE** — Device Identifier Composition Engine; layered measured-boot identity.
- **Cask** — Fjord's tamper-evident executable container.
- **Lading** — the signed manifest (bill of lading) inside a Cask.
- **DEK/VK/KEK** — data / volume / key-encryption keys in Brine's envelope.
- **Tide** — Keel's MCS scheduler; CPU time as a capability.
- **vmring** — shared-memory ring buffer for high-throughput IPC.
- **Logbook** — append-only transparency log of signatures and revocations.
